# main-starter
JVM (java / kotlin) starter using Gradle / Maven build tools.

## getting started

```bash
git clone --no-single-branch --depth=1 git@github.com:daggerok/main-starter.git
cd main-starter

git co gradle-kotlin
git co maven-kotlin
git co gradle
git co maven
git co all

rm -rf .git
# ...
```
